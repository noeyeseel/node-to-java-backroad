# node-to-java-backroad
how to study nodejs

Step 01: JS-Fundamental (자바스크립트 핵심/비동기)

Step 02: Node-Core-Deep (http 모듈, 파일 시스템, 이벤트 루프 실험)

Step 03: Express-Layered (졸업작품 기능을 쪼개서 구현 - 아키텍처 연습)

Step 04: Java-Bridge (노드로 짠 로직을 자바 문법으로 한 번 더 짜보기)



지금부터 5월 말까지 평일 3시간 + 주말 5시간 기준

[1~2주차] 자바스크립트 & 노드 근본 다지기
목표: 비동기(Promise, Async/Await) 완벽 이해.

활동: 01-javascript-basics 폴더에 비동기 로직 연습 코드 올리기.

팁: AI 쓰지 말고 console.log 찍어가며 실행 순서 메모하기.

Git 기록: "왜 노드에서 비동기가 중요한가?"를 README에 정리.

[3~4주차] 서버 원리와 Express 기초
목표: HTTP 통신 구조 파악 및 Express 서버 구축.

활동: 03-express-server 폴더에 간단한 CRUD(게시판 기초) 구현.

자바 연결: 자바의 정적 타입(Static Type) 개념을 미리 읽어보기.

Git 기록: "노드 미들웨어의 작동 원리" 기록.

[5~6주차] 졸업작품 기능 구현 & 레이어드 아키텍처
목표: 졸업작품의 핵심 기능(예: 로그인, 데이터 저장)을 하나 완성하기.

활동: 코드를 짤 때 Route - Service - Controller로 분리해서 짜기. (이게 자바 스프링의 표준 구조입니다.)

Git 기록: "코드를 분리(Layered)했을 때의 장점" 정리.

[7~8주차] Java 브릿지 & 포트폴리오 정리 (지원 준비)
목표: 노드로 짠 기능 하나를 자바로 '번역'해보고, 깃허브 전체 정리.

활동: 04-java-bridge 폴더에 자바 기초 문법으로 짠 로직 올리기.

핵심: 현장실습 자기소개서에 **"Node.js로 백엔드 원리를 깨우치고, 이를 바탕으로 Java의 견고함을 공부 중입니다"**라는 스토리 만들기.
